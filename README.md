# Futuristic Cybersecurity Developer Portfolio 🚀

A highly interactive, 3D futuristic personal portfolio website for an ethical hacker & cybersecurity enthusiast. Built with React, Three.js, GSAP, and Tailwind-inspired styling.

## ✨ Features
* **Interactive 3D Universe**: A central 3D rotating model powered by React Three Fiber.
* **Cinematic Animations**: Smooth, complex timeline animations and scroll effects via GSAP.
* **Custom Glowing Cursor**: A unique interactive cursor effect.
* **Responsive Design**: Flawless experience across all devices.
* **Component-Based Architecture**: Scalable React 18 frontend.
* **Performance Optimized**: Lazy-loaded 3D models and vendor bundle splitting.

## 🛠️ Tech Stack
* **Frontend**: React 18, TypeScript, Vite
* **3D Graphics**: Three.js, React Three Fiber, React Three Drei, Postprocessing
* **Animations**: GSAP (GreenSock), SplitText
* **Styling**: Vanilla CSS with modern custom properties and media queries
* **Icons**: React Icons (Material Design)

## 🚀 Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bhanuteja05/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   *The app will be running at `http://localhost:5173`.*

## 📦 Production Build

To create an optimized production build:
```bash
npm run build
```
This will output static files into the `dist/` directory.

## 🌐 Vercel Deployment

This project is optimized for deployment on [Vercel](https://vercel.com/). A `vercel.json` is included to correctly handle client-side routing.

1. Push your code to GitHub.
2. Go to Vercel dashboard and click **Add New Project**.
3. Import this repository.
4. **Framework Preset**: Vite
5. **Build Command**: `npm run build`
6. **Output Directory**: `dist`
7. Click **Deploy**.
